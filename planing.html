GIT Y GITHUB

Objetivo General

El estudiante conocerá y profundizara de forma dinámica, la importancia, función y manejo de las herramientas Git y GitHub para el desarrollo de Proyectos, los cuales tendran diferentes finalidades y en ellas el estudiante podra plasmar su creatividad.

Objetivos Especificos

  • Despertar en los estudiantes el interes por nuevos temas y herramientas en el mundo Tecnológico.
  • Profundizar en las herramientas Git y GitHub, que nos facilitará el trabajo en equipo para la realización de proyectos.
  • Adquirir nuevos conocimientos y habilidades, que aporten al desarrollo profesional del estudiante.
